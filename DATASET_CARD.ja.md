# Dataset Card

[English](DATASET_CARD.md) | 日本語

## 概要

この `uncommon-meter-beat-dataset` リポジトリは、変拍子と拍子変化を重視した annotation-only の beat/downbeat・拍子注釈データセットです。4/4 に強く偏った既存の public beat-tracking dataset を補完する用途を想定しています。

現在のリポジトリの集計値です。このブロックは [`parsed_beats/`](parsed_beats) から `python update_dataset_docs.py` で自動更新されます。

## 背景と目的

公開されている beat-tracking ベンチマークの多くは、奇数拍子や拍子変化をあまり含みません。そのため、標準的なベンチマークで高性能に見えるモデルでも、5/4 や 7/8、mixed meter のようなケースで大きく崩れることがあります。このデータセットは、そうしたケースを分析・評価・補強しやすくするために収集しました。

## 含まれるもの

- [`beats/`](beats) に元の `.beat` 注釈ファイル
- [`parsed_beats/`](parsed_beats) に正規化済み JSON 注釈
- [`output_graphs/`](output_graphs) にデータセット統計の図
- [`metadata.csv`](metadata.csv) にトラック単位のメタデータ

## 含まれないもの

- 音源ファイルは含みません
- 音源取得用の自動ダウンロードスクリプトは含みません
- 学習・検証・評価用の split は定義していません

各トラックは `youtube_id` で識別されます。利用者は、権利やプラットフォーム規約を確認したうえで、必要に応じて別途音源を取得する想定です。

## 注釈スキーマ

各トラックは小節単位の注釈列として表現されます。各小節には次の情報があります。

- 小節頭の時刻（秒）
- 拍子の分子と分母
- ローカルテンポ（BPM）
- テンポの基準音価
- 任意の小節単位アノテーションラベル

正規化済み JSON は、元の `.beat` ファイルを直接扱うよりも、下流の解析や変換をしやすくするための形式です。

## 収集方針

これはランダムサンプルではなく、目的特化のデータセットです。次のような条件を満たす曲を優先的に集めています。

- 5/4, 5/8, 7/4, 7/8, 9/8 などの奇数拍子を含む
- 曲中に拍子変化がある
- 既存データセットでは少ない単純拍子・複合拍子を含む

この偏りは意図的なものであり、一般的な音楽カバレッジを主張するものではなく、評価や targeted augmentation のための特徴として使うべきです。

## 想定される利用方法

- 奇数拍子・拍子変化に対する beat/downbeat モデルの評価
- 一般的でない拍子における beat-tracking システムの誤り解析
- 変拍子で性能が落ちるモデルへの補助データや fine-tuning データ
- 非標準拍子に偏らせた時間的構造の分布分析

## 既知の制約

- 大規模音源コーパスに比べると小規模です。
- 拍子分布は意図的に偏っており、自然な出現頻度を表すものではありません。
- 最終小節の長さは近似値です。
- 可読性のためファイル名には曲名が残っていますが、正規キーは `youtube_id` です。

## 法的・ライセンス上の注意

このリポジトリが配布するのは注釈とメタデータのみで、音源は含みません。リポジトリのライセンスは [MIT License](LICENSE) です。
